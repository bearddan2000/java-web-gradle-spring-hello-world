# java-web-gradle-spring-hello-world

## Description
A POC for a web app springboot java gradle build.

## Tech stack
- springboot
  - web

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
